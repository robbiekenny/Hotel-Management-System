User Manual

User Home Screen - Manage Booking Tab 1.2

    1. Choose a room from the list to add to your existing booking
    2. Back Button: Press the back button to go back a page
    3. Remove Button: Press the remove button to remove a room from the right hand list
    4. This shows the total price of rooms selected
    5. Save Changes Button: Press the save changes button to save any changes you made to your booking
    6. A list of rooms you have decided to add/remove
    7. Add Room Button: Press the add button to add a room from the left list onto the right hand list
