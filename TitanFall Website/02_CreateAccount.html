User Create Account Screen

    All fields must other than your Home Address must be less than 50 characters in length to be considered valid data
    To create an account simply fill in your correct relevant information and press continue

    2. Surname textfield
    3. Home address textfield
    4. Telephone textfield
    5. Email address textfield
    6. Username textfield
    7. Password textfield
    8. Login Button: Press the login button to go to the log in screen
    9. Continue Button: Press the continue button in order to carry onto the user home screen
    10. Help Link: Click help to access our user manual
